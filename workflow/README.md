## MCP Communication Workflow

This folder contains the step-by-step communication flow
between the user, MCP Host, MCP Server, and connected tools.

### Workflow Steps:
1. User enters a natural language prompt
2. MCP Host requests available tools
3. MCP Server returns tool definitions
4. LLM selects the appropriate tool
5. MCP Server executes the action
6. Result is returned to the user
